To Do list

To Do list app built for assignment as part of the curriculum for the JavaScript course on The Odin Project.

Restaurant site

Restaurant site built for assignment as part of the curriculum for the JavaScript course on The Odin Project.

Site has home page (slogan & working hours), menu page (dishes names and prices & dish image shown on hover and click), contact page (restaurant contact and contact form to send message) and about page (story about restaurant history).

CSS3, HTML5, JavaScript, Webpack, ESlint & Git were used in the making of this project.
